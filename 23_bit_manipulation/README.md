# Phase 23: Bit Manipulation

## Topics
- Bitwise operators (AND, OR, XOR, NOT, shifts)
- Check/Set/Clear/Toggle bits
- Power of 2 checks
- XOR tricks (find missing, find single)
- Counting set bits
- Bit masking

3. Single Number (find unique using XOR)
4. Single Number II (element appearing once, rest thrice)
5. Single Number III (two unique elements)
6. XOR of numbers in range [L, R]
7. Find missing number (XOR approach)
8. Reverse bits
9. Divide two integers without using multiplication/division
10. Subsets using bit masking
