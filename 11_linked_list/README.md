# Phase 11: Linked List

## Topics
- Singly Linked List (insert, delete, traverse)
- Doubly Linked List
- Fast & Slow pointers (Floyd's cycle detection)
- In-place reversal
- Merge operations
- Linked List as interview favorite

## Practice Questions (20)

### Easy
1. Reverse a linked list (iterative)
2. Reverse a linked list (recursive)
3. Find middle of linked list
4. Detect cycle in linked list (Floyd's)
5. Merge two sorted linked lists
6. Remove duplicates from sorted list
7. Find intersection point of two linked lists
8. Check if linked list is palindrome

### Medium
9. Find starting point of cycle
10. Remove Nth node from end
11. Add two numbers as linked lists
12. Reverse linked list in groups of K
13. Rotate linked list by K places
14. Flatten a multilevel linked list
15. Copy list with random pointer (deep copy)
16. Reorder list (L0→Ln→L1→Ln-1→...)
17. Sort linked list (merge sort)
18. Odd-Even linked list rearrangement

### Hard
19. Merge K sorted linked lists
20. Reverse nodes in K-group
